<h1> Erode ROI</h1>
<HR></HR>
<!------------------->
This script shrinks (erodes) the current ROI

    <P>	


	</P> 
	<P>The input in the dialog is the amount to shrink in each direction :
	
		<TABLE>


			<tr>
			<td> <b> Width x (pixels)</b></td>
			<td> Number of pixels in x-direction </td>
			</tr>
			

			<tr>
			<td> <b> Width y (pixels)</b></td>
			<td> Number of pixels in y-direction </td>
			</tr>
			
			<tr>
			<td> <b> Width z (pixels)</b></td>
			<td> Number of pixels in z-direction </td>
			</tr>
			


		</TABLE>


	</P> 

	<P>
		NOTE: Requires Matlab Imaging Toolbox
	</P>
