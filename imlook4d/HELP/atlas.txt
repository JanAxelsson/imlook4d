<h1>SPM atlas</h1>
<HR></HR>
<!------------------->
	<P>
		Makes region-of-interests (ROIs) from the Neuromorphometrics atlas included in SPM. 
	</P>

	<P>
		If you have multiple co-registered images in your study, the ROIs can be used on any of the co-registered images.
	</P>

	<P> 
        This script requires SPM packages to be added to the path.
	</P>

	<P> 
        The SPM package can be found at <a href="http://www.fil.ion.ucl.ac.uk/spm/software/download.html">http://www.fil.ion.ucl.ac.uk/spm/software/download.html </a>
	</P>

	<P> 
        If files are created with SPM you may be interested in comparing the opening function
        to imlook4d's normal opening functions.  Note that conversions between file formats
        is not supported by imlook4d, so that a file opened by SPM should be saved by SPM.
	</P>

	<P> 
        See also SPM write
	</P>
