<h1>Orientation drop-down list</h1>
<HR></HR>
<!------------------->
<P>
	Sets the orientation to of the imlook4d display.  This is done by changing the underlying matrix from default (which is called Axial).
</P>
<P>
	The orientation is set back to Axial before "Save", "Save ROI", "Load ROI", so that all file operations are performed in the Axial view.
</P>
