<h1>SPM Dicom to Nifti</h1>
<HR></HR>
<!------------------->
	<P>
		Converts a static or dynamic DICOM series to a 4D Nifti using SPM routines
	</P>

	<P> 
        The SPM package can be found at <a href="http://www.fil.ion.ucl.ac.uk/spm/software/download.html">http://www.fil.ion.ucl.ac.uk/spm/software/download.html </a>
	</P>

	<P> 
        Running this script, a file selection dialog is opened, where one DICOM file is selected.
        All DICOM files in the same folder will be opened, and converted to a 4D Nifti.
       The converted file is opened in imlook4d after conversion.
	</P>

	<P> 
        See also "SPM write", and "SPM open"
	</P>
